# Lextech — Technology that works. Partners you can trust.

Lextech is a B2B technology partner helping businesses in Ghana and West Africa solve real problems with practical, reliable technology solutions. We consult, build, and support, combining technical expertise with a deep understanding of how local businesses operate.

## 🚀 Services

- **Web Development**: Professional, fast websites that represent your brand and convert visitors.
- **UI/UX Design**: Clean, intuitive interfaces that make products easy to use.
- **IT Support & Solutions**: Reliable technical support and proactive maintenance to reduce downtime.
- **Custom Software**: Tailored tools and platforms built around specific business operations.
- **Cloud Workspace Setup**: Secure setup and management of Google Workspace or Microsoft 365.
- **Digital Marketing Tech**: Analytics, tracking, and automation tools for data-driven growth.

## 🛠️ Technology Stack

- **Structure**: Semantic HTML5
- **Styling**: Vanilla CSS (Modern CSS variables, Flexbox, Grid)
- **Interactions**: Vanilla JavaScript
- **Aesthetics**: Professional corporate design with dark mode support.

## 🌐 Features

- **Dark Mode Support**: Persistent theme toggle via `localStorage`.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop.
- **Micro-animations**: Subtle entrance animations and hover effects for a premium feel.
- **SEO Optimized**: Built with semantic tags and meta descriptions.
